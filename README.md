## 🚀 What is ZentraAI?

ZentraAI is the most premium AI tools directory on the internet. 106+ AI tools across 12 categories with a built-in AI chatbot (Zentra) powered by Claude API.

---

## ✨ Features

- 🔍 **Instant Search** — Find any AI tool in milliseconds
- 📁 **12 Categories** — Website Building, Image Gen, Music, Code, and more
- 🤖 **Zentra AI Chatbot** — Claude-powered assistant that recommends tools
- 🎨 **Premium Dark UI** — Three.js particles, GSAP scroll animations, Lenis
- 🖱️ **Custom Cursor** — Magnetic hover interactions
- 🔊 **Sound Design** — Immersive audio on every interaction
- 📱 **Mobile Responsive** — Perfect on all screen sizes

---

## 🛠️ Tech Stack

| Layer        | Tool                    |
|-------------|-------------------------|
| Frontend     | React + Vite            |
| Styling      | Tailwind CSS            |
| 3D / Particles | Three.js              |
| Scroll       | GSAP + ScrollTrigger    |
| Smooth Scroll| Lenis                   |
| Animation    | Framer Motion           |
| Sound        | Howler.js               |
| AI Chatbot   | Claude API (Anthropic)  |
| Routing      | React Router v6         |
| Deploy       | Vercel                  |

## ⚙️ Setup

```bash
git clone https://github.com/muhammadzamran/zentra-ai
cd zentra-ai
npm install
cp .env.example .env
# Add your Anthropic API key to .env
npm run dev
```

---

## 🔑 Environment Variables

```
VITE_ANTHROPIC_API_KEY=your_claude_api_key_here
```

Get your API key at: https://console.anthropic.com

---

## 🌍 Deploy to Vercel

```bash
npm install -g vercel
vercel
```
Add `VITE_ANTHROPIC_API_KEY` in Vercel dashboard → Settings → Environment Variables.
